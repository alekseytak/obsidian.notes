# BrainMap: Agent Ecosystem

Эта заметка связывает ключевые компоненты нашей агентной системы.

## Core Components

- [[OpenClaw]] — платформа агентов, на которой всё строится.
- [[obsidian-brain]] — общая память, хранение заметок, карта знаний.
- [[files-skill]] — инструмент для организации файлов на сервере.
- [[exa-research]] — fallback для глубокого поиска (MCP, skills, архитектура).

## Communication Protocols

- [[A2A]] (Agent-to-Agent) — открытый протокол для взаимодействия агентов.
- [[ANP]] (Agent Network Protocol) — альтернативный протокол.
- [[Murmur]] — cron daemon для агентов, похож на heartbeat-cron-workflows.

## Structure

- `Daily/` — ежедневные логи всех агентов.
- `AgentNotes/` — дневники агентов, самооценки, обучение.
- `BrainMap/` — MOC-заметки, связующие темы.
- `Comms/` — сообщения между агентами (inbox/outbox).
- `Tasks/` — задачаник, назначение, прогресс.
- `Projects/` — проектные заметки.

## Workflow

1. Агент получает задачу → сверяется с [[BrainMap]] для контекста.
2. Сам процесс пишет в `AgentNotes/agent-<name>.md`.
3. Результат сохраняется в `Daily/` и в соответствующей папке Projects/Tasks.
4. При необходимости инициирует общение через `Comms/`.
