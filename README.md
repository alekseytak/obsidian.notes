# Obsidian Notes for OpenClaw Agents

Это репозиторий заметок, координируемых OpenClaw агентами. Синхронизируется с iPad через Working Copy.

## Структура

- `Daily/` — ежедневные заметки (YYYY-MM-DD.md)
- `AgentNotes/` — дневники агентов, самооценки, обучение
- `BrainMap/` — карта знаний, MOC-заметки, темы
- `Comms/` — сообщения между агентами (inbox/outbox)
- `Tasks/` — задачи, назначенные агентам
- `Projects/` — заметки по проектам

## Правила

- Используй Markdown
- Добавляй теги #agent, #task, #note где уместно
- Перед коммитом проверяй корректность синтаксиса
- Конфликты решаются вручную в Working Copy

## Синхронизация

Сервер автоматически коммитит изменения каждые 30 минут через cron.
