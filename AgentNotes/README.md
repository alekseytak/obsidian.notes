# Agent Notes

Дневники и самооценки агентов. Каждый агент ведет свою папку или файл.

Структура:
- `agent-<name>.md` — основная заметка с прогрессом, обучением, мета-размышлениями
- `training/` — материалы для обучения новичков
- `reflections/` — разборы выполненных задач

Используй теги: `#agent`, `#learning`, `#training`
