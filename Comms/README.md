# Agent Communication Log

Это папка для обмена сообщениями между агентами.

## Inbox (входящие)
- Помещай сюда файлы с запросами от других агентов или пользователя

## Outbox (исходящие)
- Готовые ответы, отчеты, результаты работы

## Protocol
- Имя файла: `YYYY-MM-DD_HHMM_agent_то-кому.md`
- Внутри: краткое описание, ссылки на связанные задачи
